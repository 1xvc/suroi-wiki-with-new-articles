# Wiki Changelog

## January 7, 2024

This is a big update! We have been working hard on new features.

**Features**

- New generated image previews for guns, throwables and skins
- New animated preview for melees
- New gun graphs for calculating the average damage of a gun at specific ranges
- New home page
- New search and navigation UI
- New sidebar and table UI
- New sidebar image background color selector for better sprite visibility
- New search backend
- New edit on GitHub button
- New image galleries
- New buildings, throwables, backpacks and skins pages
- Art guide port

Huge thanks to Kenos for adding most of these features, and Compositr for testing and refining the code!
