The **KRISS Vector** is a SMG added in the v0.15.0 update. It shoots a fairly accurate stream of bullets rapidly, similar to the [Micro Uzi](/weapons/guns/micro_uzi), but with higher accuracy. The Vector can usually be found in the [Oil Tanker](/buildings/oil_tanker_ship_tanks)

# Strategies

## Usage

- Make sure to be careful with using the Vector, as it can consume a lot of ammo

## Countering

- The Vector consumes a lot of ammo, which means that you can dodge an opponent's bullets and force them to run out of ammo, allowing you to attack easily

# Trivia

- The Vector is another gun that is also from the game [surviv.io](https://survivio.fandom.com/wiki/Vector)
